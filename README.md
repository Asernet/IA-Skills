# Collezione di Skills IA

Questa repository contiene una raccolta organizzata di **Skills** per agenti AI, progettate per estendere le capacit√† di assistenti intelligenti come Gemini.

## Contenuto

Le skills sono organizzate in cartelle tematiche e coprono diversi ambiti, tra cui:
- **Sviluppo Software**: Inizializzazione progetti, code review, debugging sistematico.
- **Content Creation**: Copywriting, prompt engineering, SEO.
- **Gestione Conoscenza**: Obsidian, gestione documenti.
- **Data & Analytics**: Arricchimento dati, tracciamento, reportistica.

Ogni skill contiene istruzioni dettagliate (file `SKILL.md`) per guidare l'agente nell'esecuzione di compiti specifici.

## Utilizzo

Queste skills sono pensate per essere utilizzate all'interno dell'ecosistema Gemini (o compatibili), fornendo istruzioni strutturate "Just-in-Time" per migliorare la precisione e l'efficacia delle risposte.
