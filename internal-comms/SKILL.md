## Quando usare questa skill

Per scrivere comunicazioni interne, usa questa skill per:

- Aggiornamenti 3P (Progress, Plans, Problems)
- Newsletter aziendali
- Risposte FAQ
- Report di stato
- Aggiornamenti leadership
- Aggiornamenti progetto
- Report incidenti

## Come usare questa skill

Per scrivere qualsiasi comunicazione interna:

1. **Identifica il tipo di comunicazione** dalla richiesta
2. **Carica il file guida appropriato** dalla directory `examples/`:
   - `examples/3p-updates.md` - Per aggiornamenti team Progress/Plans/Problems
   - `examples/company-newsletter.md` - Per newsletter a livello aziendale
   - `examples/faq-answers.md` - Per rispondere a domande frequenti
   - `examples/general-comms.md` - Per qualsiasi altra cosa che non corrisponda esplicitamente a una di sopra
3. **Segui le istruzioni specifiche** in quel file per formattazione, tono e raccolta contenuti

Se il tipo di comunicazione non corrisponde a nessuna guida esistente, chiedi chiarimenti o pi√π contesto sul formato desiderato.
