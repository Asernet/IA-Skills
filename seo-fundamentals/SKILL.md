# Fondamentali SEO

> Principi per la visibilità sui motori di ricerca.

---

## 1. Framework E-E-A-T

| Principio         | Segnali                                    |
| ----------------- | ------------------------------------------ |
| **Esperienza**    | Conoscenza di prima mano, esempi reali     |
| **Expertise**     | Credenziali, profondità di conoscenza      |
| **Autorevolezza** | Backlink, menzioni, riconoscimento settore |
| **Affidabilità**  | HTTPS, trasparenza, info accurate          |

---

## 2. Core Web Vitals

| Metrica | Target  | Misura                  |
| ------- | ------- | ----------------------- |
| **LCP** | < 2.5s  | Performance caricamento |
| **INP** | < 200ms | Interattività           |
| **CLS** | < 0.1   | Stabilità visiva        |

---

## 3. Principi SEO Tecnico

### Struttura Sito

| Elemento      | Scopo              |
| ------------- | ------------------ |
| Sitemap XML   | Aiuta crawling     |
| robots.txt    | Controlla accesso  |
| Tag canonical | Previeni duplicati |
| HTTPS         | Segnale sicurezza  |

### Performance

| Fattore         | Impatto         |
| --------------- | --------------- |
| Velocità pagina | Core Web Vital  |
| Mobile-friendly | Fattore ranking |
| URL puliti      | Crawlability    |

---

## 4. Principi SEO Contenuto

### Elementi Pagina

| Elemento         | Best Practice                      |
| ---------------- | ---------------------------------- |
| Title tag        | 50-60 caratteri, keyword davanti   |
| Meta description | 150-160 caratteri, convincente     |
| H1               | Uno per pagina, keyword principale |
| H2-H6            | Gerarchia logica                   |
| Alt text         | Descrittivo, non stuffato          |

### Qualità Contenuto

| Fattore     | Importanza             |
| ----------- | ---------------------- |
| Profondità  | Copertura completa     |
| Freschezza  | Aggiornamenti regolari |
| Unicità     | Valore originale       |
| Leggibilità | Scrittura chiara       |

## 6. Linee Guida Contenuti AI

### Cosa Cerca Google

| ✅ Fai                     | ❌ Non Fare                    |
| -------------------------- | ------------------------------ |
| Bozza AI + editing umano   | Pubblicare contenuto AI grezzo |
| Aggiungi insight originali | Copiare senza valore           |
| Review esperti             | Saltare fact-checking          |
| Segui E-E-A-T              | Keyword stuffing               |

---

## 7. Fattori Ranking (Prioritizzati)

| Priorità | Fattore                           |
| -------- | --------------------------------- |
| 1        | Contenuto di qualità e rilevante  |
| 2        | Backlink da siti autorevoli       |
| 3        | Page experience (Core Web Vitals) |
| 4        | Ottimizzazione mobile             |
| 5        | Fondamentali SEO tecnico          |
