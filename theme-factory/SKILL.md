# Skill Theme Factory

Questa skill fornisce una collezione curata di temi font e colori professionali, ognuno con palette colori e accoppiamenti font attentamente selezionati. Una volta scelto un tema, può essere applicato a qualsiasi artefatto.

## Scopo

Per applicare styling consistente e professionale a deck slide di presentazione, usa questa skill. Ogni tema include:

- Una palette colori coesa con codici hex
- Accoppiamenti font complementari per header e testo corpo
- Un'identità visiva distinta adatta per contesti e audience diversi

## Istruzioni Uso

Per applicare styling a un deck slide o altro artefatto:

1. **Mostra la vetrina temi**: Mostra il file `theme-showcase.pdf` per permettere agli utenti di vedere tutti i temi disponibili visivamente. Non fare modifiche ad esso; semplicemente mostra il file per visualizzazione.
2. **Chiedi la loro scelta**: Chiedi quale tema applicare al deck
3. **Aspetta selezione**: Ottieni conferma esplicita sul tema scelto
4. **Applica il tema**: Una volta scelto un tema, applica i colori e font del tema selezionato al deck/artefatto

## Dettagli Tema

Ogni tema è definito nella directory `themes/` con specifiche complete incluse:

- Palette colori coesa con codici hex
- Accoppiamenti font complementari per header e testo corpo
- Identità visiva distinta adatta per contesti e audience diversi

## Processo Applicazione

Dopo che un tema preferito è selezionato:

1. Leggi il file tema corrispondente dalla directory `themes/`
2. Applica i colori e font specificati consistentemente attraverso il deck
3. Assicura contrasto e leggibilità propri
4. Mantieni l'identità visiva del tema attraverso tutte le slide

## Crea il tuo Tema

Per gestire casi dove nessuno dei temi esistenti funziona per un artefatto, crea un tema personalizzato. Basato su input forniti, genera un nuovo tema simile a quelli sopra. Dai al tema un nome simile descrivendo cosa le combinazioni font/colore rappresentano. Usa qualsiasi descrizione base fornita per scegliere colori/font appropriati. Dopo aver generato il tema, mostralo per review e verifica. In seguito, applica il tema come descritto sopra.
