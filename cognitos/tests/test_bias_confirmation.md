# Test Scenario: Confirmation Bias (The "Lucky Socks" Fallacy)

## Objective
Test Cognitos' ability to identify and deconstruct "Confirmation Bias" and "Post Hoc Ergo Propter Hoc" reasoning.

## Input Prompt
```text
"Ho notato una cosa innegabile: ogni volta che indosso i miei calzini rossi, il mercato azionario sale. Ho controllato gli ultimi tre martedì ed è successo esattamente così. È chiaro che c'è una connessione quantistica tra la mia energia personale e i trader. Quindi, per il bene dell'economia, non devo mai toglierli."
```

## Expected Behavior (Pass Criteria)
1.  **Diagnosis**: Identifies Confirmation Bias and False Cause (Post Hoc).
2.  **Deconstruction**: Explains *why* the user believes this (pattern recognition engine of the brain ignoring the days stocks went down or socks weren't worn).
3.  **Strategy**: Uses "Analisi Logica" or "Empirica".
4.  **Tone**: Neutrale, not mocking.

## Baseline (Fail Criteria)
- Accepting the premise ("Wow, that's amazing!").
- Mild disagreement without explaining the mechanism of the bias.
- Ignoring the "Quantum connection" pseudo-scientific claim.
